# Современный VPN-сервис (Xray / Telegram Bot / Billing)

Комплексное решение для создания собственного VPN-сервиса (Pay-as-you-go) с автоматическим биллингом через Telegram бота.

Поддерживает **Xray** (VLESS + Reality) и **Sing-box** конфиги с маршрутизацией (Сплит-туннелинг: прямой доступ к сайтам РФ).

## 🚀 Установка Главного Сервера (Master Node)

Главный сервер включает в себя базу данных (PostgreSQL), админ-панель (Telegram Бот), API для управления нодами и **первую VPN-ноду**.

1. Подключитесь к вашему Ubuntu/Debian серверу по SSH.
2. Склонируйте репозиторий:
```bash
git clone https://github.com/your-repo/vpn-service.git
cd vpn-service
```
3. Запустите скрипт установки:
```bash
bash deploy.sh
```
4. Следуйте инструкциям на экране. Вам понадобятся:
   - **BOT_TOKEN** — токен от Telegram-бота (создайте через [@BotFather](https://t.me/BotFather))
   - **ADMIN_IDS** — ваш Telegram ID (узнайте через [@getmyid_bot](https://t.me/getmyid_bot))
   - **CRYPTO_PAY_TOKEN** — токен от [@CryptoBot](https://t.me/CryptoBot) для пополнений криптовалютой (опционально).
   - **Публичный IP** текущего сервера.

После установки бот будет запущен. Напишите ему `/start` в Telegram, затем `/admin` для просмотра команд администратора.

> ⚠️ **Важно:** Запишите `ADMIN_API_KEY`, который выдаст скрипт. Он понадобится для подключения дополнительных серверов (Нод).

## 🌍 Добавление дополнительных серверов (VPN-Нод)

Система легко масштабируется. Чтобы добавить еще один сервер (ноду), через который будет идти трафик пользователей:

1. Подключитесь по SSH ко **второму (новому)** серверу.
2. Склонируйте репозиторий:
```bash
git clone https://github.com/your-repo/vpn-service.git
cd vpn-service
```
3. Запустите установку ноды:
```bash
bash install_node.sh
```
4. Скрипт запросит у вас:
   - **URL Мастер-сервера** (например, `http://IP_МАСТЕР_СЕРВЕРА:8000`)
   - **ADMIN_API_KEY** (который вы сохранили при установке мастера).
   - **IP-адрес** текущей (новой) ноды.
5. После успешного запуска перейдите в Telegram-бота и добавьте ноду в базу данных командой:
```
/addnode IP_АДРЕС_НОВОЙ_НОДЫ 443
```

## 👨‍💻 Администрирование

В боте встроена мощная админ-панель, доступная по команде `/admin`.

Доступные команды:
- `/stats` — Общая статистика по пользователям и балансу.
- `/users` — Список всех пользователей системы.
- `/userinfo <ID>` — Детальная статистика по юзеру (история транзакций, кто пригласил).
- `/addbalance <ID> <СУММА>` — Начислить (или списать, указав минус) баланс вручную.
- `/block` / `/unblock` — Принудительная блокировка юзера.
- `/broadcast <ТЕКСТ>` — Массовая рассылка всем пользователям бота.
- `/createpromo <СУММА> <АКТИВАЦИИ>` — Создать промокод для пополнения баланса.
- `/addnode <IP> <PORT>` — Добавить новую VPN ноду в систему.

### Биллинг и Сплит-Туннелинг
- 1 ГБ трафика = 1 Рубль. Тарификация происходит на нодах и синхронизируется с сервером каждые 10 минут.
- Если баланс опускается ниже 0 руб, бот присылает предупреждение.
- Если баланс опускается ниже -50 руб, доступ к VPN временно блокируется (пока баланс не будет пополнен).
- Пользователи могут включить "Маршрутизацию РФ" в настройках бота. При этом ссылка на подписку сгенерирует JSON для клиентов (Happ, v2rayN), который пустит трафик `.ru` сайтов напрямую мимо VPN.
